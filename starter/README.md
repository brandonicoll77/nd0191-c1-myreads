# MyReads

A bookshelf app that lets you categorize books you have read, are currently reading, or want to read. Built with React as part of the Udacity React Fundamentals course.

## Features

- View books organized across three shelves: Currently Reading, Want to Read, and Read
- Move books between shelves using the dropdown on each book card
- Search for new books and add them to a shelf
- Search results reflect the current shelf status of any book already in your library

## Getting Started

Install dependencies:

```
npm install
```

Start the development server:

```
npm start
```

The app will open at [http://localhost:3000](http://localhost:3000).

## Backend

This app uses the Udacity Books API. Search queries are limited to the terms listed in [SEARCH_TERMS.md](SEARCH_TERMS.md).
